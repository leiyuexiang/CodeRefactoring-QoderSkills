# 修复安全约束与核心原则 - 3.7.0-SNAPSHOT

## 与 3.6.x 系列的差异

本版本安全约束与 3.6.0-SNAPSHOT 基线完全一致，无差异。

## 安全约束

本版本安全约束继承 3.6.0-SNAPSHOT 基线（C-01 至 C-06、S-01 至 S-14），包含确定性保证原则、名称冲突预防、禁止语义归并、import 联动完整性、禁止重新生成文件内容（S-13）及编码保留（S-14）等约束。

> 完整安全约束请参考 3.6.0-SNAPSHOT 版本的 [safety-constraints.md](../../3.6.0-SNAPSHOT/scripts/safety-constraints.md)。
